<h1>Дипломный проект на тему "Веб-сайт по покупке билетов в кинотеатр"</h1><br>
<h2><br><b>Основные функции сайта:</b><br></h2>
<h3><br><i>Общие функции:</i><br></h3>
<p>
  <ul>
    <li>Регистрация</li>
    <li>Авторизация</li>
  </ul>
</p>
<h3><br><i>Для администратора:</i><br></h3>
<p>
  <ul>
    <li>Просмотр статистики на сегодня</li>
    <li>Просмотр статистики фильмов</li>
    <li>Просмотр статистики сеансов</li>
    <li>Просмотр сегодняшних сеансов</li>
    <li>Управление фильмами:
      <ul>
        <li>детальная информация</li>
        <li>редактирование</li>
        <li>добавление</li>
        <li>удаление</li>
      </ul>
    </li>
    <li>Управление сеансами:
      <ul>
        <li>детальная информация</li>
        <li>добавление</li>
        <li>удаление</li>
      </ul>
    </li>
    <li>Управление залами:
      <ul>
        <li>детальная информация</li>
        <li>добавление</li>
        <li>удаление</li>
      </ul>
    </li>
    <li>Обновление роли пользователя</li>
  </ul>
</p>
<h3><br><i>Для пользователя:</i><br></h3>
<p>
  <ul>
    <li>Просмотр каталога фильмов</li>
    <li>Выбор фильма</li>
    <li>Выбор сеанса на фильм</li>
    <li>Выбор места</li>
    <li>Покупка билета</li>
    <li>Просмотр купленных билетов</li>
    <li>Редактирование информации о себе</li>
  </ul>
</p>
<h2><br><b>Стек технологии:</b><br></h2>
<p>
  <ol>
    <i><li>Backend:</li></i>
    <ul>
      <li>Java Spring Boot(Spring Security, Spring Data JPA, Spring Boot DevTools, Thymeleaf, Spring Web, Spring Validation, Lombok)</li>
    </ul>
    <i><li>Frontend:</li></i>
    <ul>
      <li>HTML5</li>
      <li>CSS3</li>
      <li>Bootstrap5</li>
      <li>JavaScript</li>
    </ul>
    <i><li>Database:</li></i>
    <ul>
      <li>MySQL</li>
      <li>Docker Desktop</li>
    </ul>
  </ol>
</p>
